# magic_eden_form


Register in Google Form for MagicEden free NFTs


## Installation
Import your mails in `mails.txt`

Import your eth wallets in `eth.txt`

Import your sol wallets in `eth.txt`


Enter in terminal:

```
pip install -r requirements.txt
```
